# Contract-OS — 세움디자인하우징 전산 계약서

수기로 작성하던 ㈜세움디자인하우징 계약서를 **전산 계약서**로 만든 웹앱입니다.
원본 양식 그대로 화면에서 작성하고, 금액을 자동 계산하며, 인쇄/PDF로 출력하고, 계약 건을 저장·관리할 수 있습니다.

## 주요 기능

- **원본 양식 그대로 작성** — 계약금액 · 주문내용(8개 항목) · 약관 · 공급자/계약자 서명란을 동일한 레이아웃으로 화면에서 입력
- **금액 자동계산**
  - 평당 항목은 `평수 × 단가`로 항목 금액 자동 계산
  - 제품공급가(항목 합계) · 부가세(10%) · 제품합계 자동 산출
  - 결제 스케줄(계약금·중도금1·2·3·잔금) 합계가 제품합계와 일치하는지 실시간 안내, 한글 금액 표기
- **인쇄 / PDF 출력** — `인쇄` 버튼 → A4 가로 한 장. 입력칸 배경·테두리가 사라져 인쇄본은 깔끔한 계약서로 출력 (브라우저 인쇄에서 "PDF로 저장" 선택 시 PDF 생성)
- **계약 건 저장·관리** — 계약마다 연도별 번호(예: `2026-0001`) 자동 부여, 목록에서 건축주명·현장주소·계약번호 검색, 불러오기·수정·삭제, 작성중/확정 상태 표시

## 실행 방법

사전 준비: **Node.js 22.5 이상** (외부 라이브러리 설치 불필요 — Node 내장 기능만 사용)

```bash
npm start          # 또는: node --no-warnings server.js
```

브라우저에서 <http://localhost:3000> 접속.

> 포트를 바꾸려면: `PORT=8080 npm start`

## 데이터 저장 / 백업

- 모든 계약 데이터는 `data/contracts.db` (SQLite) 파일 하나에 저장됩니다.
- **백업**: `data/` 폴더의 `.db` 파일을 복사해 두면 됩니다.
- 이 파일은 git에 포함되지 않습니다(`.gitignore`).

## 폴더 구조

```
Contract-OS/
├── server.js          # HTTP 서버 + REST API (Node 내장 http)
├── lib/db.js          # 저장소 (Node 내장 node:sqlite)
├── public/
│   ├── index.html
│   ├── css/app.css    # 화면 + 계약서 레이아웃
│   ├── css/print.css  # 인쇄/PDF (A4) 레이아웃
│   └── js/
│       ├── model.js   # 계약 데이터 기본값(항목·단가·약관) + 자동계산
│       ├── api.js     # 서버 통신
│       └── app.js     # 목록/작성 화면, 자동계산, 저장/인쇄
└── data/              # 계약 DB 저장 위치 (백업 대상)
```

## API (참고)

| 메서드 | 경로 | 설명 |
|--------|------|------|
| GET    | `/api/contracts?q=검색어` | 목록 / 검색 |
| POST   | `/api/contracts` | 새 계약 생성(번호 자동 부여) |
| GET    | `/api/contracts/:id` | 단건 조회 |
| PUT    | `/api/contracts/:id` | 수정 |
| DELETE | `/api/contracts/:id` | 삭제 |
