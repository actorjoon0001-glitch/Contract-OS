# 이미지 자료

## 법인 인감 도장 (seal.png)

공급자(대표) 서명란에 자동으로 표시되는 **법인 인감 도장 이미지**를 이 폴더에 넣습니다.

- 파일 경로: `public/img/seal.png`
- 권장: 배경이 투명한 PNG, 정사각형(예: 500×500)
- 이 파일이 없으면 서명란에는 기존처럼 `(인)` 글자가 표시됩니다.

> 도장 이미지를 교체하려면 같은 이름(`seal.png`)으로 덮어쓰면 됩니다.
> 경로를 바꾸려면 `public/js/model.js`의 `SUPPLIER.sealImage` 값을 수정하세요.
